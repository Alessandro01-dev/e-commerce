# 🛒 E-Commerce Platform - MERN Stack Store

**E-Commerce Platform** è un'applicazione web professionale e completa per la vendita online, progettata per gestire cataloghi complessi e pagamenti sicuri.

L'integrazione di un database strutturato con 500 prodotti suddivisi in 50 categorie permette di simulare un'esperienza di shopping reale, supportata da una ricerca avanzata e una gestione fluida del carrello e dei pagamenti tramite Stripe.

---

## 🚀 Demo
L'applicazione è distribuita su Vercel: [https://e-commerce-one-eta-33.vercel.app](https://e-commerce-one-eta-33.vercel.app)

## ✨ Funzionalità Principali

- **Catalogo Dinamico**: Naviga tra **500 prodotti** organizzati in **50 macro-categorie** diverse.
- **Ricerca Avanzata & Filtri**: Sistema di ricerca istantaneo per **titolo** del prodotto e filtraggio granulare per **categoria** per trovare esattamente ciò che cerchi.
- **Wishlist Persistente**: Salva i tuoi prodotti preferiti per ritrovarli al prossimo accesso, con gestione dinamica dello stato (aggiunta/rimozione).
- **Stripe Checkout**: Integrazione completa con **Stripe API** per una gestione dei pagamenti sicura e professionale.
- **Admin Dashboard**: Pannello di controllo dedicato per la gestione dell'inventario, permettendo di **aggiungere, modificare o rimuovere** prodotti in tempo reale.
- **User Authentication**: Sistema di accesso sicuro basato su **JWT (JSON Web Token)** con protezione delle rotte lato client e server.
- **Responsive Design**: Interfaccia moderna e ottimizzata per desktop, tablet e mobile grazie a **React-Bootstrap**.

---

## 🛠️ Tech Stack

- **Frontend**: React.js (Vite), React Router Dom, React-Bootstrap, Lucide React (Icone).
- **Backend**: Node.js, Express, MongoDB (Mongoose).
- **Payments**: **Stripe SDK** per la gestione del flusso di pagamento e generazione del clientSecret.
- **Authentication**: **JWT** per il login e **JWT-Decode** per la protezione delle rotte e gestione della sessione.
- **Utility**: **React Dropzone** (upload immagini), **Cloudinary** (hosting immagini), **React Hot Toast** (notifiche dinamiche).

## 🇺🇸 English version

**E-Commerce Platform** is a professional MERN stack application featuring a dynamic catalog of 500 products across 50 categories. 
It provides a seamless shopping experience with advanced search, persistent wishlist, and secure Stripe payment integration.

### ✨ Key Features
- **Dynamic Catalog**: Browse **500 products** organized into **50 distinct categories**.
- **Advanced Search & Filtering**: Instant search by **product title** and category-specific filtering for an efficient user experience.
- **Persistent Wishlist**: Save and manage your favorite items across sessions.
- **Stripe Integration**: Secure payment processing powered by **Stripe API**.
- **Admin Dashboard**: A dedicated management tool to **Add, Edit, or Remove** products and monitor inventory in real-time.
- **Secure Auth**: Full authentication flow using **JWT** to protect user data and private routes.
- **Modern UI**: Fully responsive interface built with **React-Bootstrap**.
